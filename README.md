**JSMP** is a a Sales management software written in java. It's not really a practical piece of software. I built this while I was learning. Everything here should look messy and unoptimized. You can safely ignore this repository. 

If you are a beginner to Java it can be a great fun project for you to learn Java. Later I plan to add GUI with *JavaFX* and actually make it useful. For now it's just a CLI program.

## Key Feature Demonstration
1. OOP Concepts
2. FileIO
3. Object Serialization
4. A very low secured somewhat working authentication system without any standard.

And finally it's a fresh piece of (messy) software written by a real human not AI. None of the codes are written by AI here. Although I took help.
